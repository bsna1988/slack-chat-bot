For you, the new committer:

  1. File your ICLA and send it to secretary@apache.org
  2. Log in to https://whimsy.apache.org; that will confirm a working ASF account
    1. you can edit email routing for the account, and add other emails that you own
    2. you can directly edit mailing list subscriptions (for example, you might switch them to your ASF account - you can still post from any of your registered emails)
  3. Link your GitHub account with your ASF account at https://gitbox.apache.org/ once you see the big green "Merge" button on pull requests, this is working
  4. Read the Beam committer guide: https://beam.apache.org/contribute/committer-guide/
  5. Read the ASF new committer guide: https://www.apache.org/dev/new-committers-guide.html

For the PMC:

  1. \[Chair\] Add the new committer in whimsy
  2. Announce the new committer
  3. Add the new committer with "committer" level access in Jira
