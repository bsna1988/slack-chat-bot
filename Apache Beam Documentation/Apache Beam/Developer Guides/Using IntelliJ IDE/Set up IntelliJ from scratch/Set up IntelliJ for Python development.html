Setting up Python development is only necessary if you are working on Python. Otherwise, you can avoid Python by only building the module that you're interested in.


If you are working on Python and want to use IntelliJ, follow the steps below:

  • First ensure you've followed the most up-to-date instructions for Developing with the Python SDK, and verify that you can build Python from the commandline:
./gradlew :sdks:python:buildPython
  • Launch IntelliJ from a terminal. This ensures that the necessary virtualenv environment is configured:
/opt/intellij-ue-stable/bin/idea.sh
  • In IntelliJ, open the Gradle window, and select beam | sdks | python | Tasks | other | buildPython.

The build should succeed without any errors.

Remove generated files from indexing

Open project view, right click ~/sdks/python/target/.tox → Mark Directory As → Excluded
