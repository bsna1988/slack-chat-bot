The Beam repository is composed of many many modules, and running a full build will be very time consuming. We recommend building and testing a single module that you're working on.

If you would like to build the entire codebase, ensure that you've set up your IntelliJ environment and followed all language-specific instructions:

  • Set up IntelliJ for Python development


  1. In your IntelliJ beam project, navigate to View | Tool Windows | Gradle to open the gradle sidebar
  2. In the gradle sidebar, find the top-level "beam (root)" project and expand it
  3. Under the project, expand Tasks | build, and double-click the build task to execute it.
